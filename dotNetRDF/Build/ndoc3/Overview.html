dotNetRDF API Documentation

Overview

dotNetRDF is a C#/.Net Library for working with RDF and the Semantic Web which is entirely free and open source.

This is the formal API documentation for the Library whose main namespace is VDS.RDF. You may alternatively wish to look at the User Guide which contains informal documentation and examples or just visit the main Project Website.
